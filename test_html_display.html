Ejemplos de Visualización HTML en el Chatbot

Esta página contiene ejemplos de elementos HTML que ahora se pueden mostrar correctamente en el chatbot.

Ejemplo de Tabla

Copia y pega el siguiente código en el chatbot:

```html
<table class="table table-striped">
  <thead>
    <tr>
      <th>Producto</th>
      <th>Descripción</th>
      <th>Precio</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Producto 1</td>
      <td>Descripción del producto 1</td>
      <td>$100.00</td>
    </tr>
    <tr>
      <td>Producto 2</td>
      <td>Descripción del producto 2</td>
      <td>$150.00</td>
    </tr>
    <tr>
      <td>Producto 3</td>
      <td>Descripción del producto 3</td>
      <td>$200.00</td>
    </tr>
  </tbody>
</table>
```

Resultado esperado:

Producto Descripción Precio
Producto 1 Descripción del producto 1 $100.00
Producto 2 Descripción del producto 2 $150.00
Producto 3 Descripción del producto 3 $200.00

Ejemplo de Lista

Copia y pega el siguiente código en el chatbot:

```html
<h3>Lista de tareas pendientes:</h3>
<ul>
  <li>Completar informe mensual</li>
  <li>Enviar correos a clientes</li>
  <li>Actualizar sitio web</li>
  <li>Programar reunión semanal</li>
</ul>
```

Resultado esperado:

Lista de tareas pendientes:

  • Completar informe mensual
  • Enviar correos a clientes
  • Actualizar sitio web
  • Programar reunión semanal

Ejemplo con Imágenes y Enlaces

Copia y pega el siguiente código en el chatbot:

```html
<div>
  <h3>Recursos útiles:</h3>
  <p>Visita <a href="https://www.ejemplo.com">nuestro sitio web</a> para más información.</p>
  <p>También puedes consultar la <a href="https://www.ejemplo.com/documentacion">documentación</a>.</p>
</div>
```

Resultado esperado:

Recursos útiles:

Visita nuestro sitio web para más información.

También puedes consultar la documentación.
